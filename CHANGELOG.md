# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),

## [Unreleased]

### Added
- Dark/light/system theme toggle via `#themeBtn` button, persisted in `chrome.storage.session` (#15)
- Inline SVG icons (lucide-style) for search, clear, and theme toggle buttons (#15)
- Full-width Exact Match button below the search input (#15)

### Changed
- Complete flat UI redesign matching Figma specification (#15)
- Primary accent color updated from indigo `#4854c8` to blue `#2563eb` (blue-600) (#15)
- Popup width increased from 400px to 500px (#15)
- CSS custom properties for full light/dark theme support (#15)
- Font Awesome CDN removed in favour of inline SVGs (#15)
