Features

Everything you need to tame tab chaos

No more hunting through dozens of tabs. Tab Search is fast, keyboard-first, and stays out of your way.

⚡

Real-time search

Results update instantly as you type — no delay, no button to press. Searches both tab titles and URLs.

⌨️

Keyboard navigation

Arrow keys to move, Enter to switch tab, Escape to close. Never touch the mouse if you don't want to.

🌙

Dark mode

Automatically follows your system preference. Supports light and dark themes with WCAG AA contrast.

♿

Accessible

Full ARIA support, screen reader compatible, minimum 4.5:1 contrast ratio on all text elements.

🗂️

Tab management

Switch to any tab, close individual tabs, or close all others with a single click. Confirmation for bulk actions.

🔒

Privacy first

No external servers. No analytics. All processing happens locally in your browser. Open source.

Installation

Get started in 60 seconds

Manual installation is quick. Chrome Web Store publishing is coming soon.

Download the extension

Grab the latest .zip from GitHub Releases and unzip it to a folder.

Open Chrome Extensions

Navigate to chrome://extensions in your browser.

Enable Developer Mode

Toggle the Developer mode switch in the top-right corner of the Extensions page.

Load the extension

Click Load unpacked and select the unzipped folder. Tab Search will appear in your toolbar immediately.

Usage

How it works

Click the Tab Search icon in your Chrome toolbar (or assign a keyboard shortcut in chrome://extensions/shortcuts) and start typing.

🔍

Search

Type any word from the tab's title or URL. Matching text is highlighted. Use the exact-match toggle for precise results.

🖱️

Switch

Click any result or press Enter on the keyboard-selected row to jump directly to that tab — even across different windows.

✂️

Close

Hit the × button on any row to close that tab, or use Close others to keep only the selected tab open.
